# CATERPILLAR-PLATFORM
Гусеничная платформа на Arduino управляемая по Bluetooth
1. В наличии для прототипирования:
* 1 х гусеничная платформа
* 1 х Motor Shield TB6612
* 1 х модуль bluetooth SPP-C
